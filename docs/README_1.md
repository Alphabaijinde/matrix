## Map Selection

| Map ID | Map Name     | Description |
|--------|--------------|-------------|
| 0      | CustomWorld  | Custom map |
| 1      | Warehouse    | Warehouse scene (SceneWorld) |
| 2      | Town10World  | Town scene |
| 3      | YardWorld    | Courtyard scene |
| 4      | CrowdWorld   | Crowd scene with 50 randomly walking pedestrians and obstacles |
| 5      | VeniceWorld  | Venice-style town with rivers, bridges, and corridors |
| 6      | HouseWorld   | Three-bedroom apartment scene |
| 7      | RunningWorld | Coin collection game |
| 8      | Town10Zombie | Zombie game |
| 9      | IROSFlatWorld | IROS competition map 1 (flat terrain) |
| 10     | IROSSlopedWorld | IROS competition map 2 (sloped terrain) |
| 11     | IROSFlatWorld2025 | IROS competition map 3 (flat terrain 2025) |
| 12     | IROSSloppedWorld2025 | IROS competition map 4 (sloped terrain 2025) |
| 13     | OfficeWorld  | Two-story office building scene |
| 14     | 3DGSWorld    | 3D Gaussian Splatting Reconstruction Map |
| 15     | MoonWorld    | Moon Surface Simulation Map, Dynamic Ground Surface |

> **Note:** EmptyWorld is the default map included in the base package and configured via `DefaultEngine.ini`. It is not run via map ID, but serves as the engine's default startup map.
